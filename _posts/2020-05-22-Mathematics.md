# 基础数学学习

## 数系的介绍

* N (Natural) - **自然数**
    * { 1, 2, 3, 4, 5, ... }
* Z (Integer) - **整数**
    * { ..., -3, -2, -1, 0, 1, 2, 3, ... }
* Q (Quotient) - **有理数**
    * { X | X = Q/P P.QεZ P≠0 }
* R (Real) - **实数**
