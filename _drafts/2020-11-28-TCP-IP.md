# TCP / IP

## 起源

60 年代末，美国政府资助的一个分组网络研究项目。90 年代，已经发展成为计算机之间最常见应用的组网形式。

## 分层结构

### 数据链路层

相关协议：

* ARP（Address Resolve Protocol）
* RARP（Reverse Address Resolve Protocol）

### 传输层

相关协议：

* TCP
* UDP

### 网络层

相关协议：

* IP
* ICMP
* IGMP

### 应用层

相关协议：

* Telnet
* FTP
* CMTP
* SNMP
